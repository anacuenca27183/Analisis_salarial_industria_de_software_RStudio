# Analisis_salarial_industria_de_software_RStudio
Este proyecto presenta un análisis estadístico, generado con RStudio, del sector del desarrollo de software utilizando la base de datos de la encuesta de desarrolladores de Stack Overflow 2025.  El objetivo es comprender cómo varían los salarios según: el país, tipo de puesto, nivel educativo, años de experiencia, modalidad laboral e industria..
